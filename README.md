# 💳 Flutter Wallet App

A modern digital wallet application built with **Flutter** and **Dart** that provides users with a clean financial dashboard to view wallet balances, monitor transaction history, and manage their digital wallet experience. The project demonstrates reusable widgets, modular architecture, responsive UI design, and clean Flutter development practices.

---

# 📱 Overview

The Flutter Wallet App simulates a digital wallet where users can view their balance, browse recent transactions, and navigate through a modern financial interface. The project focuses on creating reusable UI components and maintaining a scalable architecture for future enhancements.

## 📂 Project Structure

```text
lib/
│
├── components/
├── config/
├── widgets/
├── assets/
└── main.dart
```

### 📁 Folder Description

**components/**
Contains reusable UI components such as:

* Wallet Balance Card
* Transaction Tile
* Dashboard Sections
* Custom Buttons

**config/**
Stores application-wide configurations including:

* Colors
* Themes
* Constants
* App configuration values

**widgets/**
Contains reusable widgets shared across multiple screens to improve maintainability and reduce duplicate code.

**assets/**
Stores images, icons, fonts, and other application resources.

**main.dart**
Entry point of the application that initializes the app and loads the home screen.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-wallet-app.git
```

### Navigate to the project

```bash
cd flutter-wallet-app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
